> 变化都是表面的东西，内在的东西其实并没有太多的变化。
> 掌握技术基础可以让自己找到答案和知识，基础是抽象和归纳，很容易形成进一步的推论。

## 分类：

* 程序语言：语言的原理，类库的实现，编程技术（并发、异步等），编程范式，设计模式……
* 系统原理：计算机系统，操作系统，网络协议，数据库原理……
* 中间件：消息队列，缓存系统，网关代理，调度系统 ……
* 理论知识：算法和数据结构，数据库范式，网络七屋模型，分布式系统……

## 学习效率

1. 挑选一手知识和信息源
2. 要注重基础原理
3. 使用知识图谱
4. 学会举一反三
5. 总结和归纳
6. 实践和坚持
